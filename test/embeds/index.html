Embed Options

Add options in the head of your webpage. Ensure it is set immediately and not after the page has loaded (don't use inside document.onReady or equivalent).

lang

  • 'en' (ENGLISH)
  • 'fr' (FRENCH)
  • 'de' (GERMAN)
  • 'es' (SPANISH)
  • 'zh' (CHINESE)
  • 'ru' (RUSSIAN [limited support])
document.GW2A_EMBED_OPTIONS = {
  lang: 'en',
};

Styling Embeds

To style any embed simply target the embed with the embed name, e.g:

.gw2a-character-embed
.gw2a-items-embed
.gw2a-skills-embed
.gw2a-tooltip-embed
.gw2a-specializations-embed

Character

  • data-armory-embed="character"
  • data-armory-name="{CHARACTER_NAME}"
<div
  data-armory-embed="character"
  data-armory-name="Blastrn"
>
</div>

<script async src="https://gw2armory.com/gw2aEmbeds.js"></script>

Skills

  • data-armory-embed="skills"
  • data-armory-ids="{IDS_COMMA_SEPARATED}"
  • data-armory-{ITEM_ID}-stat="{ITEMSTAT_ID}"
<div
  data-armory-embed="skills"
  data-armory-ids="5507,5508,5510,5515,5516,5517,30662,5542,79712"
  data-armory-79712-stat="1153"

</div>
